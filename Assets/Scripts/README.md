# Project Structure

This document outlines the new, reorganized structure of the `Assets/Scripts` directory.

## Directories

*   **Data**: Contains scriptable objects and data-related scripts.
*   **Managers**: Contains high-level manager scripts that control game-wide systems.
*   **Powerups**: Contains scripts related to player power-ups.
*   **Systems**: Contains scripts that manage specific game systems.
*   **UI**: Contains scripts related to the user interface.
