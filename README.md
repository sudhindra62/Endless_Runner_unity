# 🏃 Endless Runner Game (Unity 6.3 LTS)

An Android endless runner game built with **Unity 6.3 LTS (6000.3.2f1)**.

Inspired by:
- Subway Surfers
- Temple Run

This project is designed with a **modular architecture**, scalable systems, and long-term production stability in mind.

---

## 🎯 Project Goals

- Infinite stable gameplay
- Subway Surfers / Temple Run–style mechanics
- Modular feature expansion
- Monetization-ready architecture
- Firebase integration
- Zero compile errors
- Long-session gameplay stability (hours without crash)

---

## 🛠 Tech Stack

- Unity 6.3 LTS
- C#
- Android (Play Store target)
- Firebase (Analytics + Remote Config)
- Google Mobile Ads (AdMob – planned monetization)
- GitHub (CI/CD ready)

Primary: Android (Google Play Store)

## 🚧 Current Status

🚧 Active development

## ⚠️ Development Rules

- Do **NOT** introduce duplicate movement systems.
- Do **NOT** override forward speed directly.
- Maintain modular architecture.
- Avoid deprecated Unity APIs.
- Ensure zero compile errors.
- Gameplay must remain stable for infinite sessions.
- Core gameplay must never break while adding features.
